# Landing Page – Bordados

Landing page de vendas para bordados artesanais.

## Arquivos

- `artmariae.html` — página principal (HTML, CSS e JS em arquivo único)
- Logo do cliente incluída na pasta

## Para adaptar para um novo cliente

- Substituir a logo na seção Hero
- Atualizar nome da marca no topo e no rodapé
- Inserir número do WhatsApp nos 6 links `wa.me`
- Atualizar nomes, descrições e preços dos produtos
- Ajustar cores em `:root` no CSS se necessário
